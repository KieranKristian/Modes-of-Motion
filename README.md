Controls:
WASD - Movement
Space Bar - Jump
E - turn robot off/on
Left mouse click - interact with objects when robot is on

Added Packages:
Unity Starter Assets
Probuilder
Text Mesh Pro

Features Included:
Game level modelled to look like a spaceship interior
Cannon which also shakes the camera and plays a noise when missile is fired
Control panel, allowing you to change variables of a cube that is being constantly lerped (can turn off position,rotation,scale or colour, can increase duration, can change easing) all using UI elements on world space camera
Generator with balls bouncing using rigidbody, bouncy physics material and add force constantly
Danger reading using hinge joint which has constant force applied to make it wobble

Lerping:
Position Lerping
Rotation Lerping
Scale Lerping
Object Color/Text Colour/Light Colour Lerping

Curves:
Bezier Curves
Cardinal Spline

Maths Library:
Lerp equation functions for float, vector2, vector3 and vector4's
Curve equations: Bezier and Cardinal spline
Power equation
Square root function

Easings Library containing large variety of easings to use

Sound Effects:
Explosion sound Effec: used for when missile is fired https://freesound.org/people/jobro/sounds/35460/
Beep Effect: Played when robot is turned on, played when objects are interacted with and played when any UI element is affected
